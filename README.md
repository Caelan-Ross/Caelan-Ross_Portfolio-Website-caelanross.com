# Caelan Ross – Portfolio Website

Personal portfolio website built with React.js and MUI. Features a dark theme, multi-page layout, and a contact form with direct email delivery.

**Live Site**: [https://caelanross.netlify.app](https://caelanross.netlify.app)

---

## Pages

### Home
- Hero section with name and title
- About Me card with background, experience, and education
- Core skills preview (5 highlighted technologies) linking to the full Skills page
- Featured project card (most recent) linking to the full Projects page

### Skills
- Full categorized skills list across Frontend, Backend, Infrastructure, Networking, Cloud, Dev Tools, Dev Practices, and AI-Assisted Development
- Each skill displays calculated years of experience

### Projects
- Two-column card grid of professional and personal projects
- Each card shows an overview, tech stack chips, and an expandable technical details section
- Public projects include a direct repo link

### Contact
- Contact form (Name, Email, Subject, Message) with validation
- Sends messages via EmailJS
- Sidebar with phone, email, and GitHub contact info

---

## Tech Stack

- **Framework**: React.js (Create React App)
- **UI Library**: MUI v5 (dark theme)
- **Form Handling**: react-hook-form + EmailJS
- **Routing**: React Router v6
- **Icons**: MUI Icons

## Getting Started

### Prerequisites

- Node.js v14 or newer
- npm

### Installation

```bash
git clone https://github.com/Caelan-Ross/Caelan-Ross_Portfolio-Website-caelanross.com.git
cd Caelan-Ross_Portfolio-Website-caelanross.com/app
npm install
```

### Running Locally

```bash
npm start
```

Visit `http://localhost:3000`.

### Build

```bash
npm run build
```
